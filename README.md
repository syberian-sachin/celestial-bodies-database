# Celestial Bodies Database

This project is a PostgreSQL relational database that stores information about celestial bodies such as galaxies, stars, planets, and moons. It was created as part of the freeCodeCamp Relational Database Certification.

## Technologies Used
- PostgreSQL
- SQL

## Project Features
- Relational database design
- Primary and foreign key constraints
- Structured data for galaxies, stars, planets, and moons
- Database dump included in `universe.sql`

## File
- `universe.sql` — PostgreSQL database dump containing the complete database schema and data.
